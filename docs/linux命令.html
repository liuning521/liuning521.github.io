    # 安装Cent-OS7系统

    • 插入系统盘，进入BIOS引导界面

    • install cent os

    • 如果安装过程中出现了Starting dracut情况

    • 先cd /dev ,然后ls查看安装盘对应的设备号 （一般是sdb4）

    • 重新进入到引导安装界面，然后按e进入到启动引导编辑状态

      vmlinuz initrd=initrd.imginst.stage2=hd:LABEL=CentOS\x207\x20x86_64 rd.live.check quiet
改为：
vmlinuz initrd=initrd.imginst.stage2=hd:/dev/sdb4 quiet

    • 然后按ctrl x保存并进入安装界面

    • 安装可顺利进行

    # 安装Cent-OS8

    • 通过镜像地址下载Cent-OS8 (opens new window)

      为了避免踩坑，此处建议下载将近9G左右的Cent-OS介质

    • 使用rufus (opens new window)以dd的方式将介质写入到U盘当中

      使用rufus以dd的方式将介质写入到U盘当中，可以避免系统安装的时候，选不到安装源的问题

      使用UltraISO将介质写入到U盘当中，会有不少坑（如果能保证安装系统的时候，系统能连接到外网则可避免问题），不建议采用

    • 重启电脑进入到BIOS引导界面，按照指引操作

    # 放开Cent-OS的端口号

    # 查看所有放开的端口
firewall-cmd --list-ports

# 查看某端口号是否已经开启
firewall-cmd --query-port=8889/tcp

# 新增开启的端口号 --permanent 长期开启
firewall-cmd --add-port=8889/tcp --permanent

# 重载入添加的端口，新增开启的端口号后徐执行此命令
firewall-cmd --reload
